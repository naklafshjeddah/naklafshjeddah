# شركة الأفضل لنقل العفش بجدة | Al Afdal Moving Company Jeddah

<div dir="rtl">

موقع إلكتروني احترافي متكامل لشركة نقل عفش في جدة مبني بـ Next.js 14 مع دعم كامل للغتين العربية والإنجليزية.

</div>

A complete, production-ready moving company website built with Next.js 14 featuring full Arabic/English bilingual support.

---

## 🌟 Key Features | المميزات الرئيسية

### 📱 Multilingual Support | دعم متعدد اللغات
- ✅ **Arabic (default)** at `/` - **العربية (الافتراضية)** على `/`
- ✅ **English** at `/en` - **الإنجليزية** على `/en`
- ✅ Seamless language toggle - تبديل سلس بين اللغات
- ✅ RTL/LTR support - دعم الاتجاهات اليمنى واليسرى

- 🏠 **1** Homepage - صفحة رئيسية

- 🚚 **10** Intercity route pages - صفحات مسارات بين المدن
- 📦 Storage, Contact, Offers pages - صفحات التخزين والاتصال والعروض

**Total: 94+ pages with full Arabic & English content!**

## 🚀 Quick Start | البدء السريع

### Prerequisites | المتطلبات
- Node.js 18+ 
- npm or yarn

### Installation | التثبيت

```bash
# Clone the repository
git clone https://github.com/naklafshjeddah/naklafshjeddah.git

# Navigate to directory
cd naklafshjeddah

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

افتح [http://localhost:3000](http://localhost:3000) في المتصفح.
